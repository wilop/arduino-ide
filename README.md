# Arduino IDE
 
Arduino IDE 2.x: The open-source software for programming Arduino boards (binary version).

## URL:
- [Arduino IDE 2.x](https://www.arduino.cc/en/software/#ide "https://www.arduino.cc/en/software/#ide")  
- [Git Repository](https://github.com/arduino/arduino-ide "https://github.com/arduino/arduino-ide")

## Installation:  
To install `arduino-ide`, run the following command:  

```bash
kcp -i arduino-ide
```
